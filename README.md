# shast LAB - GAS Web App

ラボ用バーコードスキャン入力 + サブコードQRラベル生成スクリプト。

## 構成

| ファイル | 役割 |
|---|---|
| `Code.gs` | サーバー側ロジック (スキャン受付、シート書込、QR生成) |
| `Index.html` | ラボタブレットで開く入力UI |

## 使い方

### 1. 前提シート
スプレッドシート内に以下のシートが存在すること:

- **バーコード**: shast が書き込む採取記録 (A=地点 B=上下 C=コード D=採取時刻)
- **工程時刻**: A〜D列はバーコードシートからARRAYFORMULAで同期、E〜I列がGAS書込先
  - 1行目ヘッダー: `地点 / 上下 / コード / 採取 / 受入 / 風乾 / 振動 / 濾過 / 分析`
  - A2: `=ARRAYFORMULA(IF(バーコード!A2:A="","",バーコード!A2:A))`
  - B2: `=ARRAYFORMULA(IF(バーコード!B2:B="","",バーコード!B2:B))`
  - C2: `=ARRAYFORMULA(IF(バーコード!C2:C="","",バーコード!C2:C))`
  - D2: `=ARRAYFORMULA(IF(バーコード!D2:D="","",バーコード!D2:D))`
- **バーコード作成**: 受入時にQRラベルが追加される (空でOK、自動初期化)

### 2. GAS設定

1. スプレッドシートを開く
2. 「**拡張機能 → Apps Script**」
3. デフォルトの `Code.gs` の中身を全部削除して、本リポジトリの `Code.gs` を貼り付け
4. ファイル一覧で「+」→「**HTML**」→ 名前 `Index` で作成
5. `Index.html` の中身を貼り付け
6. 保存 (Ctrl+S)

### 3. Web App デプロイ

1. 右上「**デプロイ → 新しいデプロイ**」
2. 種類: 「ウェブアプリ」を選択
3. 設定:
   - 説明: `shast LAB v1`
   - 次のユーザーとして実行: **自分**
   - アクセスできるユーザー: **自分のみ** (社内運用なら「Google アカウントを持っているユーザー」)
4. 「デプロイ」
5. 表示された Web App URL をラボタブレットでブックマーク

### 4. バーコード作成シート初期化 (任意/初回)

スプレッドシートに戻ると上部メニューに「**shast LAB**」が追加される。

「**shast LAB → バーコード作成シート初期化(A-one 28379)**」をクリックすると、列幅・行高がA-one 28379(21面 70×42.3mm)に近い見た目で整形される。

## 運用

### ラボタブレット側

1. ブラウザでWeb App URLを開く
2. モードボタン (受入/風乾/振動/濾過/分析) を1回タップ
3. USBバーコードリーダーでジップロックを次々スキャン
4. スキャンするたびに該当列に時刻記録 + 画面に結果表示
5. 工程切替時のみモードボタンを切替

### バーコードルール

| サブコード | 使用場面 | スキャンするモード |
|---|---|---|
| `0001` (枝番なし) | 現場採取・ラボ受入 | 受入 |
| `0001-1` | ラボ1工程 | 風乾 / 振動 / 濾過 |
| `0001-2` | ラボ2工程 | 分析 |

### QR印刷

1. 「バーコード作成」シートには受入スキャンするたびに2ラベル(-1, -2)が追加される
2. 1日分溜まったら「**ファイル → 印刷**」
3. 用紙 A4、A-one 28379 専用ラベル用紙にセットして印刷
4. 必要に応じて余白を調整 (印刷プレビューで位置確認)
5. 切り取ってジップロックに貼付

## トラブルシュート

### 「外部URLからデータを取得するためのアクセスを許可」
スマホ/タブレットで `=IMAGE()` が `#REF!` になる場合、PCブラウザで一度スプレッドシートを開いてQR画像セルをクリック → 「アクセスを許可」をクリック。一度承認すれば以降スマホでも表示される。

### スキャンしても反応しない
- バーコードリーダーが HID(キーボード)モードか確認
- 入力欄にカーソルがあるか確認 (画面タップでフォーカス)
- 末尾にEnterが送られる設定になっているか (説明書参照)

### コードが見つからない
工程時刻シート C列にそのベースコードが存在しないと記録不可。先に shast 側でバーコードシートに採取記録を作る必要あり。

## 設計メモ

- 同じ物理バーコード `0001-1` を風乾→振動→濾過 で**使い回し**、モード選択で書き込み先列を切替
- 「分析」だけは別バーコード `0001-2` (ラボ2移送時に貼付)
- 受入スキャン時に自動でQR2枚生成 (-1 ラボ1用 / -2 ラボ2用)
